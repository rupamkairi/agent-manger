# AGENTS.md

This repo is a simple Deno Desktop + Svelte + Vite + TypeScript + Tailwind + shadcn-svelte starter.

## Where things live

- `plans/setup.plan.md`
- `package.json`
- `deno.json`
- `components.json`
- `src/frontend`
- `src/desktop`

## Work rules

- Keep changes small.
- Reuse the existing Svelte/Tailwind/shadcn-svelte setup.
- Avoid extra abstractions or new dependencies unless the plan requires them.

## Checks

- `pnpm check`
- `pnpm build`
- `deno fmt`
- `deno lint`
